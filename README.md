Sales Data Analysis Project

## Overview
This project analyzes sales data to identify trends, top-performing categories, and regional performance.

## Tools Used
- Python (Pandas, Matplotlib)
- Jupyter Notebook
- Excel
- Power BI (Dashboard)

## Project Structure
- 'sales_analysis.ipynb' – Data cleaning, analysis, and visualization
- 'sales_data.csv' – Sales dataset

## Key Insights
- Technology category contributes the highest sales
- Regional sales performance varies significantly
- Data-driven insights support business decision-making

## Conclusion
This project demonstrates end-to-end data analysis skills suitable for entry-level Data Analyst roles.
